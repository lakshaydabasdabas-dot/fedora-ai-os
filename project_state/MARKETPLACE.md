# Fedora AI OS - MARKETPLACE.md

Last updated: 2026-06-11

## Environment Marketplace
The Environment Marketplace is the discovery and distribution layer for Fedora AI OS. It enables community-driven development of "Environment Packs," allowing users to instantly equip their OS with specialized professional capabilities.

## Features
- Registry Integration: Direct link to installed environments in the Agent Registry.
- Community Packs: Developers can create packs consisting of specific skill bundles, workflow templates, and memory schemas.
- Versioning: Environment packs follow Semantic Versioning.
- Verification: Trusted packs (community-vetted) receive security badges.

## Impact on Roadmap
Long-term: Post-MCP Gateway and core workflow stability, the focus shifts to defining the Environment Pack schema and building the ingestion pipeline for the Marketplace.
